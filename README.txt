Tubes-IFD-1127050034-1127050131
===============================

Aplikasi Kesundaan Adalah aplikasi berbasis android untuk memperkenalkan 
kebudayaan Sunda kepada dunia serta bertujuan untuk membangun kesadaran untuk melestarikan kebudayaan lokal. 
Aplikasi di bangun oleh IFD-1127050034-Desy Fadliaty A ; IFD-1127050131-Rifazulrizqa M, 
yang di peruntukan pengguna umum dan semua umur.
